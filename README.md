# Bundling C demo

It's using https://github.com/xtuc/holyc for the compilation, It will likely **only work for Linux**.

## Usage

Install:
```sh
npm install
```

Webpack:
```sh
webpack -w
```

Web server:
```sh
http-server .
```

and go to [http://localhost:8080/dist](http://localhost:8080/dist/).
